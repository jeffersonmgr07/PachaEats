# Pacha Eats V1 Marketplace

Versión reorganizada del proyecto para avanzar hacia una aplicación tipo Rappi / Uber Eats.

## Qué incluye

- Home para cliente final.
- Catálogo de platos y restaurantes.
- Carrito en localStorage.
- Checkout simulado en soles.
- Seguimiento de pedido simulado.
- Paneles separados para clientes, restaurantes, repartidores y administrador.
- Datos centralizados en `assets/data/*.json` para reemplazo futuro por API.
- Documentación de roadmap, modelo de negocio, API y pagos.

## Cómo abrir

Abrir `index.html` en navegador o subir la carpeta completa a GitHub Pages / hosting estático.

## Próxima etapa recomendada

Conectar backend real con roles, base de datos, autenticación y pasarela de pago.
